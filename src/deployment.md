# Deployment

There are many ways to deploy an axum-based application. In this section we'll try to cover a few options.

I personally use two services to run my servers, but that's just my personal taste.

* [Linode](https://linode.com/) now owned by [Akamai](https://www.akamai.com/)
* [Digital Ocean](https://www.digitalocean.com/)

I usually run [Ubuntu](https://ubuntu.com/) on my servers. Given that I use Ubuntu as my desktop, this makes the transition smoother.
